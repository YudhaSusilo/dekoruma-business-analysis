# Dekoruma Business Analysis

## Project Overview
This project analyzes Dekoruma's sales data to uncover key business insights and identify opportunities for growth. The analysis focuses on transaction patterns, product performance, and store-level contributions to revenue.

---

## Objectives
- Understand overall sales performance
- Identify top-performing products and categories
- Analyze store contribution to total revenue
- Discover actionable insights to improve business performance

---

## Dataset
The analysis uses three main datasets:

- **Orders** → transaction-level data  
- **Products** → product details and attributes  
- **Stores** → store-level information  

All datasets are stored in: data/raw

---

## Tools & Technologies
- Python (Pandas, NumPy)
- Jupyter Notebook
- Data Visualization (Matplotlib / Seaborn)

---

## Analysis Workflow
1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Aggregation & Business Metrics  
4. Insight Generation  

Main analysis can be found in: notebooks/Dekoruma.ipynb
